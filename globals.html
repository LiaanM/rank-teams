Const rank

  • rank(input: string): Team[]
    • Defined in lib/rank.ts:44

    Converts a set of game results into a ranked table and returns the results. Input is expected in the following format

     Lions 3, Snakes 3
 Tarantulas 1, FC Awesome 0

    Example (es module)

     
    import { rank } from 'rank-teams'
const ranked = rank(`Lions 3, Snakes 3\nTarantulas 1, FC Awesome 0`);
console.log(const)
// => [
// => {name: "Tarantulas", points: 3},
// => {name: "Lions", points:1},
// => {name: "Snakes", points:1},
// => {name: "FC Awesome", points:0}
// => ]

    Example (commonjs)

     
    var rank = require('rank-teams').rank;
const ranked = rank(`Lions 3, Snakes 3\nTarantulas 1, FC Awesome 0`);
console.log(const)
 // => [
 // => {name: "Tarantulas", points: 3},
 // => {name: "Lions", points:1},
 // => {name: "Snakes", points:1},
 // => {name: "FC Awesome", points:0}
 // => ]

    Parameters

    • input: string

      String containing lines for each game: "{team1_name} {score},{team2_name} {score}\n"

    Returns Team[]

    Returns array of objects containing name and points
